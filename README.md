A beginner-friendly JavaScript project that creates dynamic toast notifications similar to modern web applications.

The application allows users to click a button and instantly display styled notification messages on the screen. Each notification is generated dynamically using JavaScript, styled with CSS, and automatically removed after a few seconds.

This project focuses on core frontend development concepts such as DOM manipulation, event listeners, dynamic element creation, functions, arrays, conditionals, timers, and UI interactions.

Features
Dynamic toast notifications
Different notification types:
Success
Error
Warning
Automatic fade-in animation
Automatic removal after 3 seconds
Random notification messages
Responsive notification container
Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)

Learning Outcome

This project helps build a strong understanding of how interactive web applications dynamically create and manage user interface components using JavaScript.
